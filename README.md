# LangChain01

LangChain experimentation and agent architecture sandbox.

Focus: tool routing, memory patterns, evaluation harnesses, and reliability testing.

## Standards
- Every strategy/change must include a reproducible validation path.
- No merges with failing CI.
- Prefer explicit risk controls and measurable acceptance criteria.

## Getting Started
1. Create a feature branch.
2. Add changes + tests/docs.
3. Open PR with validation evidence.
